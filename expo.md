# イベント機能の新規追加

## 1. チームプランの新設

- 店舗のプランに、チームプランを追加（plan_id=4）
- 価格は0円で決済なし。
- ただし、店舗側の設定画面→プラン管理画面には、一切公にせず、チームプランはあくまでもまだ、テーブルの手修正によって店舗の割り当てるのみとする。
- チームプラン契約店舗は、以下詳細記載の新規機能「イベント」管理を利用可能とする。
  （管理画面のサイドバー上、オンライン予約の上に「イベント管理」としてメニュー追加する）

## 2. 既存の機能改修部分

- (全てのプラン共通) スタッフに、”肩書き”(position) 文字列・NULL許容・inputフォーム入力を想定 の、項目を追加し、既存のスタッフ管理画面上で設定可能とする（設定なしでもOK）

## 3. イベント管理機能

- 完全なる機能追加。

- 概要：これは、各企業のオンライン出展イベントをToruya公式側で主催する際に利用する想定。類似のイメージとしては、以下のITトレンドEXPOに該当する。
  https://it.expo.it-trend.jp/
  イベントを作成すると、公開ページに各オンライン出展ブースの一覧と内容が閲覧できるフロントが用意され、作成したイベント管理画面上で、各ブース（＝コンテンツ）の内容を入れ込み、表示する。
  ユーザは、そこで概要を知った上で、イベント参加に進み、Toruya公式LINEの友だち登録と初回のプロフィール情報の入力を済ませることで、イベント参加登録が完了する。
  完了後、各コンテンツへのセミナー参加へ進める形式とする。

- イベント参加登録時に、ユーザー属性の取得・保存を行う。
  - イベント参加時のLINEログイン完了時点で以下を自動取得・照合する
    - Toruyaユーザーか否か（`user_id` の有無）
    - 無料プランか有料プランか
    - LINE連携完了済みか否か
- イベント参加登録時に、ユーザへ求める初回プロフィール登録内容
  イベント参加時のLINEログイン後に、以下3点を入力してもらう
  （全て任意選択/入力）

1. 事業内容
   - 業種選択（複数選択可）
     - セラピスト
     - 整体師
     - ネイリスト
     - アイリスト
     - Yoga講師
     - ピラティス講師
     - 美容師
     - スクール講師
     - その他（inputフォーム）
2. 開業歴
   - 選択肢（radiobox）
     - 1年未満
     - 1〜3年
     - 3年以上
3. 今の一番の悩み（選択式・単一選択）
   - 選択肢一覧（UIに表示するラベルのみ）
     **集客・認知**
     - 新規のお客様がなかなか増えない
     - SNSやホームページを頑張っているのに予約に繋がらない
     - 紹介だけに頼っていて、自分で集客する方法が分からない
       **LINE・デジタルツール活用**
     - LINEを導入したが使いこなせていない
     - LINEはメッセージ送受信にしか使えていない
     - ホームページやSNSの見た目・デザインをもっとよくしたい
       **コンテンツ・発信**
     - 発信したいことはあるのにうまく言葉にできない
     - ブログや文章を書くのが苦手で続かない
     - 集客のための文章や資料をどう作ればいいか分からない
       **経営・売上**
     - 予約は入っているのに売上が安定しない
     - 単価を上げたいが、どうすれば良いか分からない
     - 確定申告や税金・お金の管理が不安
     - 売上はあっても手元にお金が残らない
       **時間・仕組み化**
     - 集客・事務作業に時間がかかりすぎて施術に集中できない
     - リピーターが少なく、毎月集客し直しになっている
     - 予約管理や顧客対応の仕組みをもっと整えたい
       **その他**
     - その他（自由記述）
   - 🔧 エンジニア向け仕様：悩み選択肢とおすすめ出展者業種のマッピングテーブル
     **目的：**
     イベント後に弊社から送信するフォローアップLINEMessageで、お悩みに対応する出展者のコンテンツ（展示ブース・セミナー）を優先的に案内するために使用する。
     **データ保存先：** `user=2584`の顧客情報備考欄に`concern_category`（カテゴリ名）と`concern_label`（選択したラベルをそのまま）を保存
     | 選択肢（concern_label） | カテゴリ（concern_category） | 出展者業種（recommended_roles） |
     | ----------------------------------------------------- | ---------------------------- | --------------------------------- |
     | 新規のお客様がなかなか増えない | acquisition | LINE集客コンサル, SNS集客コンサル |
     | SNSやホームページを頑張っているのに予約に繋がらない | acquisition | WEBデベロッパー, SNS集客コンサル |
     | 紹介だけに頼っていて自分で集客する方法が分からない | acquisition | LINE集客コンサル, セミナー講師 |
     | LINEを導入したが使いこなせていない | line_tool | LINE集客コンサル |
     | LINEはメッセージ送受信にしか使えていない | line_tool | LINE集客コンサル |
     | ホームページやSNSの見た目・デザインをもっとよくしたい | line_tool | デザイナー |
     | 発信したいことはあるのにうまく言葉にできない | content | ライター |
     | ブログや文章を書くのが苦手で続かない | content | ライター |
     | 集客のための文章や資料をどう作ればいいか分からない | content | ライター, デザイナー |
     | 予約は入っているのに売上が安定しない | management | 経営コンサル, セミナー講師 |
     | 単価を上げたいがどうすれば良いか分からない | management | 経営コンサル |
     | 確定申告や税金・お金の管理が不安 | management | 税理士 |
     | 売上はあっても手元にお金が残らない | management | 税理士, 経営コンサル |
     | 集客・事務作業に時間がかかりすぎて施術に集中できない | efficiency | LINE集客コンサル, WEBデベロッパー |
     | リピーターが少なく毎月集客し直しになっている | efficiency | LINE集客コンサル, SNS集客コンサル |

     | その他（自由記述） | other | （属性別案内の対象外） |

- コンテンツのタイプ
  - 以下の2タイプあり。
    - セミナー(動画)
    - 展示ブース(PDF)
      以下の導線。（既存オンラインサービスの流用）
      画像カルーセル→利用開始btn→PDFダウンロードサービス(出展者自身のアカウントで作成)登録URLからLINEログイン→ダウンロードページへ遷移
    - セミナーは、大元のイベント主催の店舗側アカウントで作成し、そこへ誘導。
    - 展示ブースは、出展企業側のアカウントへLINEログイン・友だち登録 → そのオンラインサービスへ誘導。

- 各コンテンツの管理内容
  - どの店舗配下のサービスか。
  - 上記店舗のうち、どのオンラインサービスへの誘導にあたるのか。
  - そのサービス利用登録の上限数。
  - コンテンツタイトル
  - サービス開始日時・終了日時
  - 説明文
  - 紹介文
  - コンテンツのサムネイル画像
  - そのサービスの資料(PDF)プレビュー向けのスライダー掲載画像（複数）
  - (セミナー動画の場合のみ) 直接DLできる資料ファイル
  - アップセル向け
    - その店舗のどの無料相談予約（オンライン予約）に誘導するか。
      - あり/なし を設定可に。ありの場合は、誘導先を指定可に。
      - ※各企業（店舗）側でオンライン予約枠を作成している前提。
    - モニター情報（ユーザからのモニター応募を受け付けたい場合）
      1. あり/なし
      2. サービス名
      3. モニター金額
      4. モニター上限(人数)
      5. 応募フォーム：GoogleフォームURL

- セミナー動画の詳細
  - 基本、セミナー動画視聴機能の既存の流用ではあるが、以下の流れのように、本編を、指定の広告動画（YouTubeかGoogleドライブのどちらかの想定）でサンドし、自動で、広告→本編→広告の再生が遷移されるように。
    　 - サムネ→利用開始btn→広告動画表示→セミナー講演動画表示→広告動画表示
  - コンテンツタイプがセミナー動画であっても、資料DLは可能とする（ただし、その場合は、出展企業へのアカウントへの誘導ではなく、直接DLできる想定）
- 特別出演者
  - セミナー動画の特別出演者は、イベント主催大元の店舗側のスタッフ登録を行うことで、フロント側で紹介・表示を行えるようにする。
    以下がスタッフ管理の範疇で設定できるように。
  1. プロフィール写真
  2. スタッフ肩書き
  3. スタッフ氏名
  4. プロフィール文
